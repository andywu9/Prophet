Prophet

Prophet came to be as a project for Software Design and Documentation, a course taught at Rensselaer Polytechnic Institute. For this course, groups would have to design a project of their choice. The HODLers, the name of our group, decided to do a project around cryptocurrency since it was a hot topic at the start of the project. Our initial idea centered around making a website where users could log in, have access to their crypto, and trade with the many different exchanges. This idea became a little above our heads when we were discussing the privacy and issues of holding users many different coins and money.

This led us to pivot to the current Prophet that we have today, a predictive algorithm for the crypto world using machine learning. Andy Wu and Adam Lawrence were already very familiar with the crypto world while Mike Adams, Igor Carvalho, and Jeremy Torella had limited experience but were open to learning.

Why Open Source

We decided to create this project with the hopes that it would further the progress in the field of machine learning, as it relates to cryptocurrency. We believe that the advancement in this field would be most efficient if we allowed the website to be open source. As an open source project, this website becomes a platform for others to test their ideas regarding predictive ML algorithms and allows a community to form around predicting cryptocurrency price changes.

More information on the code can be found in our GitHub page. https://github.com/andywu9/Prophet

Algorithm

Prophet currently uses two algorithms for price forecasts: linear regression and an LSTM, long short-term memory, neural network. A linear regression is a line that best fits the prices between a starting price and an ending price. We use linear regression to forecast trend directions which helps users determine when to buy and when to sell their cryptocurrencies. An LSTM neural network is a variant of an RNN, recurrent neural network, and is a model for short-term memory which can last over a long period of time. It allows us to chain outputs from one step of our neural network, as inputs, to the next step of our neural network. The ability to connect previous information to our current task is crucial for making better predictions. For more about how an LSTM neural network works, here is a website for a more detailed explanation.

How To

On the home page, we give you all of our information and data, but accessing all of that information may not be so clear. Here is how to get all that Prophet has to offer:


Searching

To search, simply type the name of the coin you are interested in searching for in the search bar.


Opening a pop up

Click on any specific coin and the pop up for that coin will appear.


View specific data

With the pop up open, click on a specific time range you wish and hover over the line on the graph.


Learning more

To learn more about a coin, click on the "Info" tab in the pop up window.
